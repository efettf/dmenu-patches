# fuzzymatch

## Description

Add support for fuzzy-matching to dmenu, enabling matching using non-consecutive
portions of items.

Add option `fuzzy` to `config.def.h` and the flag `-F` to toggle fuzzy-matching.

## Notes

- Supports dmenu's case insensitive switch (`-i`)
